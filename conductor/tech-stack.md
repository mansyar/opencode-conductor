# Technology Stack

## Primary Language
- **TypeScript**: Primary language for the plugin development
- **JavaScript**: Runtime environment for Node.js

## Runtime & Build
- **Node.js**: Runtime environment
- **TypeScript Compiler (tsc)**: Build tool for TypeScript compilation

## Package Management
- **pnpm**: Package manager (replacing npm as requested)

## Testing & Quality
- **Vitest**: Testing framework
- **semantic-release**: Automated versioning and release management

## Development Tools
- **TypeScript**: Type-safe JavaScript development
- **Vitest**: Fast unit testing framework
- **semantic-release**: Automated semantic versioning
